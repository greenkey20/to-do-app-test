# My ToDo App
ToDo 리스트 관리를 위한 웹 애플리케이션입니다.

## Features
- ToDo 항목에 우선순위 부여 및 우선순위에 따른 정렬이 가능합니다.
- Spring Boot로 쉽게 서버 배포가 가능합니다.
- 완료된 혹은 미완료된 ToDo 항목만 별도 조회 가능합니다.
